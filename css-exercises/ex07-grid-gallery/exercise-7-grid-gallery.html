Exercise 7

This is Exercise #7, lecture on CSS Grid, used in the course IDATA2301 Web technologies at NTNU, campus Aalesund.

Purpose

The purpose of the exercise is to practice the implicit behaviour of CSS grids: automatically-generated columns and rows.

Instructions

Use this document as a template. You see a set of images below. Configure the following behaviour for the gallery container:

  1. The images should be in a flexbox container.
  2. Each image should be 200px wide and 150px high.
  3. The container should fill the whole width of the page, with 50px margins on the sides.
  4. The container should not shrink to less than 500px horizontally (hint: min-width property).
  5. The container should be 800px high.
  6. The images should be laid out horizontally in a grid, placing as many images as possible in every row.
